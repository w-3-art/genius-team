📝 Description
Les débutants absolus passent par Cortex (app desktop, pas de terminal). Les devs passent par le CLI. Cortex lit les mêmes fichiers (state, playgrounds, logs) et spawne Claude Code en arrière-plan. Pas 2 produits séparés — 2 interfaces sur le même moteur.

🎯 Impact attendu
On adresse les non-devs (qui n'ouvrent pas un terminal) sans diluer l'expérience dev. Aucun concurrent n'a d'app desktop. C'est un différenciateur unique.

📝 Description
Mois 0-2 : 100% gratuit. Mois 3 : lancer Pro ($29/mois) avec tous les packs, templates avancés, GitHub Action, team mode. Mois 5 : Enterprise ($149/seat) avec Agency mode, SSO, audit trail. Mois 7 : marketplace templates/extensions (30% commission). Objectif : $50K MRR à 12 mois.

🎯 Impact attendu
Pérennité du projet. Un mainteneur solo ne peut pas tenir sans revenus. Le free core construit la communauté, Pro capture les power users, Enterprise capture les agences.
