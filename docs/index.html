FAQ & Troubleshooting
Common questions

‚ùì I get "command not found: claude"

Claude Code CLI isn't installed. Run: curl -fsSL https://claude.ai/install.sh | bash then restart your terminal.

‚ùì I get "permission denied"

The script needs execute permission. Try: chmod +x scripts/setup.sh or run with bash scripts/setup.sh

‚ùì What is a terminal?

It's the app where you type commands. On Mac: search "Terminal". On Windows: use WSL or Git Bash. On VS Code: press Ctrl+` to open it.

‚ùì How do I open the terminal in my project?

In VS Code: File ‚Üí Open Folder ‚Üí select your project ‚Üí then Ctrl+` (backtick) to open terminal. Or right-click the folder in Finder/Explorer and "Open in Terminal".

‚ùì The command says "my-project" but I want a different name

Replace "my-project" with YOUR project name! Example: ...create.sh) booking-app. No spaces, use dashes.

‚ùì Claude seems stuck or keeps repeating

Run /guard-check to see if something went wrong. If needed, /guard-recover gets you back on track.

‚ùì I closed the terminal and lost my progress

No worries! Just cd back to your project folder, run claude, then /continue. The memory system remembers everything.

‚ùì Do I need to pay for Claude?

Yes, Genius Team uses Claude Code which requires a Claude Pro or API subscription. Check claude.ai/pricing.

‚ùì Can I use this with GPT/Gemini/other models?

Genius Team is built for Claude Code. Some modes (Omni) can use secondary providers, but the core requires Claude.

‚ùì How do I update to a new version?

Run /genius-upgrade inside your project, or use the upgrade command from the terminal.
