What Cortex does

One brain, every project.

Multi-project dashboard

Status, health, active skill, next task — for every Genius Team repo you have.

Shared memory

Save a decision or a pattern once. Every project learns from it.

Behaviors & rules

Define what Claude Code should always do, or never do, across all your repos.

Genius Store (MCP)

Skills are served on-demand via MCP — no bundling, no duplication, always up to date.

Scheduler & triggers

Nightly QA, weekly audits, health checks — configure once, run forever.

Vault

Store API keys globally or per project. Never hardcoded, never committed.
