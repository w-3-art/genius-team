✦ GETTING STARTED

Up and running in 60 seconds

No accounts. No subscriptions. No credit card. Just one command.

Install Claude Code

Genius Team runs on top of Claude Code, Anthropic's AI coding agent. Install it with npm install -g @anthropic-ai/claude-code.

Create your project

One command creates everything: bash <(curl -fsSL genius.w3art.io/install) my-app. That's it — your AI team is ready.

Tell it what to build

Open Claude Code and say what you want. "Build me a SaaS for managing restaurant reservations." The team takes over from there.

Watch it come to life

42 specialists collaborate: the interviewer asks the right questions, the architect designs the system, the developers build it, and QA tests everything.
