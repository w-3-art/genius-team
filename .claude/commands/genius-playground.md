# /genius-playground

Generate a context-aware HTML playground dashboard for your project.

Reads your project's actual data (specs, architecture, market analysis, dev progress, SEO scores)
and creates a custom interactive dashboard — not a generic template.

## Usage

```
/genius-playground
```

## What it generates

- `PROJECT-DASHBOARD.html` — Master dashboard with all available project data
- Or a type-specific dashboard if only one data source is available

## Different from /genius-dashboard

- `/genius-dashboard` → Opens the static template playground hub
- `/genius-playground` → Generates a project-specific dashboard from YOUR real data

⚠️ **NEVER run a local script directly. Always use the slash command.**
